---

## 📌 Overview

**A.C Currencies Converter** is a clean, scalable, and user-friendly web application designed for global currency conversion.

Users can:
- Select a **continent**
- Choose a **country**
- Automatically view the **official currency with symbol**
- Convert any amount to **any other world currency**

This project is designed for real-world usage, learning, and professional deployment.

---

## ✨ Features

- 🌍 All continents supported
- 🏳️ Country selection by continent
- 💱 Convert **any currency to any other currency**
- 💲 Full currency names with symbols (₦, $, €, £, ¥)
- 🔁 Swap currencies instantly
- 📱 Fully responsive design
- 🎨 Clean and professional UI

---

## 🌎 Supported Continents

- Africa  
- Asia  
- Europe  
- North America  
- South America  
- Oceania  

Each continent includes its complete list of countries and official currencies.

## 🛠️ Technologies Used
